## Stack

Built with [Jekyll](https://jekyllrb.com/) + [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) theme, deployed via GitHub Pages.

## Run Locally

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.
