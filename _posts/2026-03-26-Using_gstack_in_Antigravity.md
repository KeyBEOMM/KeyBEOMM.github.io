antigravity 또한 cluade code와 같이 skills 기능을 지원한다. 

저자는 가난한 대학생이기에 토큰을 무지막지하게 사용하는 cluade code 대신 antigravity를 사용한다.

antigravity의 gemini 3 pro 모델로도 이 gstack을 사용하고 싶어 antigravity에 gstack을 이식해보고자 했다.

claude code의 경우, skills들을 
~/.claude/skills/ 아래에 저장한다.

antigravity의 경우, skills들을 
~/.gemini/skills/ 아래에 저장한다.
